# Skills 集合

这是一个 AI Skills（自定义指令模板）的集合仓库。

## 快速开始

```bash
# 使用 npx 快速创建 Agent Skill
npx create-agent-skills my-skill

# 交互模式
npx create-agent-skills

# 完整参数
npx create-agent-skills my-skill -d "我的 Skill" -a "developer" -t full
```
